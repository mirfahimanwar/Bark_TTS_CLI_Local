# MANUAL INSTALL (GPU — recommended):
#   Step 1:  pip install torch torchaudio --index-url https://download.pytorch.org/whl/cu124
#   Step 2:  pip install -r requirements.txt
#
#   NOTE: torch MUST be installed first with --index-url (not --extra-index-url).
#         Using only pip install -r requirements.txt will install the CPU-only
#         PyPI wheel (torch 2.10+ exists on PyPI and outranks the CUDA build).
#
# MANUAL INSTALL (CPU only — very slow, not recommended for Bark):
#   pip install torch torchaudio
#   pip install -r requirements.txt
#
# CUDA version guide — change cu124 in the Step 1 command above:
#   CUDA 12.4+  →  cu124   (default, works with most modern NVIDIA GPUs)
#   CUDA 12.1   →  cu121
#   CUDA 11.8   →  cu118
#   Check: nvidia-smi (CUDA version shown in top-right)
#   Full wheel list: https://download.pytorch.org/whl/torch/
# ─────────────────────────────────────────────────────────────────────────────

# ── PyTorch (GPU build) ───────────────────────────────────────────────────────

#       ensure the CUDA build is chosen over the CPU-only PyPI wheel.
#       To install manually with GPU support:
#         pip install torch torchaudio --index-url https://download.pytorch.org/whl/cu124
#       Then run:  pip install -r requirements.txt
torch>=2.1.0
torchaudio>=2.1.0

# ── Bark core dependencies ────────────────────────────────────────────────────
# (from suno-ai/bark pyproject.toml)
encodec>=0.1.1              # Meta's neural audio codec — required by Bark
funcy>=1.17                 # Functional utilities — imported directly in generation.py
huggingface-hub>=0.14.1     # Model weight downloading from HuggingFace
numpy>=1.23.0
scipy>=1.9.0                # wav file writing + softmax in generation
tokenizers>=0.13.0          # Fast tokenizer backend
transformers>=4.31.0        # BertTokenizer used in generation.py

# ── Audio playback (optional but recommended) ─────────────────────────────────
sounddevice>=0.4.6          # pip install sounddevice  (also needs PortAudio)
soundfile>=0.12.1           # Reading/writing audio files

# ── Voice cloning (--clone flag) ─────────────────────────────────────────────
# No extra packages needed. Uses transformers.HubertModel (facebook/hubert-base-ls960)
# which is already pulled in by the transformers dep above.
# Model weights (~360 MB) + tokenizer (~24 MB) are downloaded on first --clone run
# and cached by HuggingFace in ~/.cache/huggingface/ and voices/.hubert_cache/.

# ── Model download fallback (optional) ───────────────────────────────────
# boto3 is listed in Bark's pyproject.toml for legacy S3 checkpoint downloads.
# Modern Bark uses HuggingFace hub exclusively — boto3 is not needed in practice.

# boto3>=1.26.0

#   Bark will automatically download ~5 GB of model weights to:
#   Windows: C:\Users\<you>\.cache\suno\bark_v0\
